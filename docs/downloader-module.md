# 下载器模块

下载器模块用于配置 qBittorrent、Transmission、Aria2 等下载器，并为种子检索和自动订阅提供推送能力。

![下载器](assets/downloader.png)

## 下载器列表

下载器页展示下载器名称、地址、类型、上传/下载速度、版本、I/O、自动下载开关、连通状态、种子数、空间信息和操作按钮。

页面支持：

- 按下载器类型筛选。
- 按连通状态筛选。
- 新增下载器。
- 设置实时刷新频率。
- 立即刷新运行状态。
- 测试连接。
- 编辑配置。
- 删除配置。

## 支持类型

| 类型 | 连通测试 | 实时状态 | 种子推送 | 说明 |
| --- | --- | --- | --- | --- |
| qBittorrent | 支持 | 支持 | 支持 | 使用 Web API 登录并复用短期会话。 |
| Transmission | 支持 | 暂未展示完整实时状态 | 支持 | 使用 `/transmission/rpc`。 |
| Aria2 | 支持 | 暂未展示完整实时状态 | 支持 | 使用 JSON-RPC。 |
| 自定义 | 支持通用 HTTP 连通测试 | 不支持 | 不支持 | 可用于记录和测试自定义地址。 |

## 配置建议

- 下载器地址填写 Web API 可访问的基础 URL。
- 如果下载器有账号密码，需同步填写用户名和密码。
- qBittorrent 建议确认 WebUI 已启用，并允许系统所在网络访问。
- Transmission 需要确认 RPC 地址和认证策略正确。
- Aria2 需要确认 JSON-RPC 可访问。
- 下载路径可在自动订阅或推送时指定；不指定时使用下载器默认路径。

## 与其他模块的关系

- 种子检索：搜索结果可手动推送到下载器。
- 自动订阅：订阅命中后可自动推送到绑定下载器。
- 看板总览：下载器统计会展示下载器数量、在线数量、实时速度和累计流量。
