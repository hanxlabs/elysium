# PT 模块功能

PT 模块负责维护站点源、站点账号配置、站点数据、检索方案和种子检索，是 Elysium 的核心业务模块。

![站点数据](assets/pt-data.png)

## 站点源

站点源用于维护系统支持的 PT/BT 站点基础信息，包括站点标识、站点名称、站点类型、解析模板、主地址、签到地址、备用地址、历史地址、标签、时区、图标、代理和 FlareSolverr 开关等。

初始化站点列表见 [支持站点列表](supported-sites.md)。

## 站点配置

站点配置用于给当前用户绑定具体站点账号。常见配置项包括站点、账号名称、Cookie、状态和备注。任务和检索模块会基于这些配置访问站点页面。

建议配置后先执行站点数据刷新，确认 Cookie、站点地址、解析规则和权限都正常。

## 站点数据

站点数据页用于集中查看各站点账号状态，包含用户名、UID、等级、上传量、下载量、分享率、做种数量、做种体积、魔力值、做种积分、时魔、活动时间等指标。

页面支持：

- 按站点筛选。
- 按做种量、上传量、下载量等字段排序。
- 批量刷新。
- 一键打开当前站点。
- 信息脱敏开关。
- 同步站点。
- 查看详情和单站刷新。

## 数据检索

数据检索包含检索方案与种子检索。检索方案用于定义参与搜索的站点与搜索参数；种子检索根据方案、关键词、页码和每页数量聚合多个站点结果。

![种子检索](assets/search-torrent.png)

种子检索支持：

- 按检索方案选择站点集合。
- 输入关键词并跨站搜索。
- 按站点筛选结果。
- 按促销标签筛选结果。
- 按发布时间等字段排序。
- 展示种子大小、做种、下载、完成数。
- 标记已下载状态。
- 获取站点详情链接与下载链接。
- 将种子推送到已配置下载器。

## 解析配置

系统支持通用解析模板和站点专用解析配置。通用模板包括 `NexusPHP`、`Gazelle`、`Unit3D`、`TBDev`、`GazelleModern` 等；部分站点通过 `pt_site_parser_config_*.sql` 维护专用字段解析。

当站点页面结构变动时，优先维护站点专用解析配置；只有通用规则失效范围较大时，再调整通用模板。
