# 自动订阅模块

自动订阅模块用于把“定时检索、关键词筛选、下载器推送、通知提醒”串成一条自动化流程，适合追更番剧、剧集、纪录片或其他周期性资源。

![自动订阅](assets/auto-subscribe.png)

## 日历视图

自动订阅首页采用月历视图，按日期展示订阅计划。不同状态会以不同颜色呈现，方便观察本月订阅分布、执行状态和缺口。

页面支持：

- 上月、今天、下月切换。
- 刷新当前视图。
- 新增自动订阅。
- 点击日期查看当天订阅计划。

## 新增订阅

新增订阅时需要配置：

- 订阅名称。
- 检索方案。
- 下载器。
- 下载路径。
- 订阅时间类型：每周几、固定日期或日期区间。
- 开始时间。
- 包含关键字。
- 排除关键字。
- 通知渠道和通知方式。
- 启用或停用状态。

## 筛选逻辑

包含关键字支持 `AND` 和 `OR` 两种模式；排除关键字也支持 `AND` 和 `OR`。通常建议：

- 包含关键字用于指定清晰目标，例如片名、分辨率、季集信息。
- 排除关键字用于剔除不需要的版本，例如枪版、低清、特定字幕组或不想要的编码。
- 检索方案控制参与搜索的站点集合，关键词控制结果命中范围。

## 执行链路

自动订阅执行时会按计划调用检索方案，筛选候选种子，选择命中的结果，推送到下载器，并写入订阅记录。若配置了 Bark 通知，会在成功或失败时发送通知。

建议先用种子检索页面验证关键词和检索方案，再创建自动订阅，这样可以减少误下和漏下。
